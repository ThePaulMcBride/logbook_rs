# Logbook

Logbook is a simple command line tool to keep the records of your fligts. It is written in Rust and uses SQLite as a database. All date is stored locally on your computer.

## Demo

![Demo](/.github/demo.gif)

## Usage

### Add a new flight

```bash
logbook add
```

### List all flights

```bash
logbook list
```

## Help

```bash
logbook help
```
